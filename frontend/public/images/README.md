# Image Directory

Drop your images into the appropriate folders. The website will automatically detect and display them.

## Folder Structure

- **hero/** - Main hero/banner images for the home page
- **about/** - Personal photos, workspace photos, profile pictures
- **carbuild/** - GR86 build photos, modifications, progress shots
- **gallery/** - General gallery images

## Naming Conventions

For best results, name your files descriptively:

### Hero Images
- `hero-main.jpg` - Primary hero image on home page
- `hero-banner.jpg` - Alternative banner image

### About Images
- `profile.jpg` - Your profile photo
- `workspace.jpg` - Your development setup
- `coding.jpg` - Action shot coding

### Car Build Images
- `gr86-full.jpg` - Full car shots
- `engine-bay.jpg` - Engine modifications
- `wheels-closeup.jpg` - Wheel/tire details
- `interior.jpg` - Interior modifications
- `exhaust.jpg` - Exhaust system
- `lighting.jpg` - LED/lighting mods
- `before-after-*.jpg` - Before/after comparison shots

### Gallery Images
- Any general images you want to showcase

## Supported Formats

- JPEG/JPG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp)
- GIF (.gif)

## Image Optimization Tips

1. **Resolution**: 1920x1080 or higher for hero images, 800-1200px width for detail shots
2. **File Size**: Keep under 500KB when possible for faster loading
3. **Aspect Ratio**: 16:9 for hero images, 4:3 or 1:1 for detail shots
4. **Format**: Use JPEG for photos, PNG for graphics with transparency

## How It Works

The website automatically scans these folders and displays images:
- Images in `hero/` appear on the home page
- Images in `about/` appear on the About page
- Images in `carbuild/` appear on the GR86 Build page
- Images in `gallery/` appear in a dedicated gallery section

Just drop your images and refresh the page!
